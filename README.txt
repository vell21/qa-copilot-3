# AI QA Tool (TypeScript)

An AI-powered QA tool for generating concise test cases and QA actions from Azure DevOps (VSTS) tickets and pull requests using OpenAI.

## Features

- Fetches ticket Description, Acceptance Criteria, and PR info from Azure DevOps
- Uses OpenAI to generate test cases or QA suggestions
- REST API, ready to extend or plug into web UI
